# 前端代码规范

## 命名规范

- 组件使用 PascalCase：`UserProfile.vue`
- 工具函数使用 camelCase：`formatDate()`
- CSS 类名使用 BEM：`block__element--modifier`

## Git 规范

提交信息遵循 [[后端规范]] 中定义的通用格式。

## 技术栈

- React 18 + TypeScript
- 状态管理：Zustand
- 样式：Tailwind CSS

参见 [[前端架构]] 了解项目结构。
