# API 规范

## RESTful 接口

- URL 小写 + 连字符：`/api/v1/user-profiles`
- HTTP 方法：GET / POST / PUT / DELETE
- 统一响应格式

## 错误码

| 错误码 | 含义 |
|--------|------|
| 40001 | 参数错误 |
| 40101 | 未认证 |
| 40301 | 无权限 |
| 50001 | 服务内部错误 |

## 鉴权

JWT Token，过期时间 2 小时，刷新 Token 7 天。
