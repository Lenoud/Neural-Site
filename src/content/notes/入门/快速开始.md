# 快速开始

欢迎使用团队知识库。

## 如何写文档

1. 在对应部门目录下创建 `.md` 文件
2. 添加 frontmatter（title 必填）
3. 使用 [[链接]] 引用其他文档
4. Git push 后自动构建部署
